# Estructura de Componentes - Sistema de Estadística Hospitalaria

Esta carpeta está organizada de acuerdo con las mejores prácticas monolíticas de Next.js para un crecimiento ordenado y escalable:

- `ui/`: Componentes básicos reutilizables (Botones, Inputs, Modales, Badges de estado).
- `layout/`: Componentes estructurales de la aplicación (Menú de navegación, barra superior, diseño responsivo).
- `dashboard/`: Componentes de negocio específicos para el panel de control de KPI e indicadores.
- `charts/`: Adaptadores e integraciones personalizadas de Recharts con guardas de hidratación para Next.js.
- `forms/`: Envolturas y campos de entrada controlados e integrados con React Hook Form y Zod.
